# brass.js
hehehe doot doot
<br>
this is a trumpet you can use in a web browser!!!!!! (spiritual successor to trombonline) <br>
supports 10 partials (B♭2 to E♭5) <br>
also supports every key! (default is B♭) <br>
allows for standard controls (keyboard on desk) and raised controls (keyboard in front of you like a trumpet) <br>
tuning system toggle has returned as well! <br><br>



made using vscode (html, javascript, css) <br>
link to play: [https://azzyblooms.github.io/brass.js/](https://azzyblooms.github.io/brass.js/)
